## 🎯 סקירה כללית

**MatafOS** היא מערכת דשבורד מתקדמת המיועדת לניהול ומעקב אחר:
- ⏰ שעות עבודה של עובדים
- 📊 דרישות ותקציבים
- 👥 צוותים וביצועים
- 📈 ניתוח והצגת נתונים בזמן אמת

המערכת מאפשרת ייבוא קבצי Excel, ניתוח אוטומטי של הנתונים, והצגה ויזואלית ברורה עם אפשרות סינון לפי צוותים.

## ✨ תכונות עיקריות

### 📂 ניהול נתונים
- **ייבוא קבצי Excel** - תמיכה מלאה בקבצי XLSX/XLS/CSV
- **עיבוד אוטומטי** - ניתוח וחישוב אוטומטי של כל הנתונים
- **סינון לפי צוותים** - תצוגה ייעודית לכל מנהל צוות
- **חיפוש גלובלי** - חיפוש מהיר עם autocomplete

### 📊 תצוגות ודוחות
- **3 טאבים מרכזיים**:
  - 👥 עובדים - מעקב אחר ביצועים ושעות
  - ⏱️ דיווחי שעות - ניתוח מפורט של שעות עבודה
  - 📋 דרישות - ניהול תקציבים ודרישות

### 📈 KPIs וניתוח
- **אחוזי השקעה והוצאה** - חישוב אוטומטי עם צביעה חכמה
- **זיהוי חריגות** - התראות על חריגות תקציב
- **סיכומים מפורטים** - לכל עובד, צוות ודרישה

### 🎨 ייצוא ודוחות
- **ייצוא ל-Excel** - ייצוא מלא של כל הנתונים
- **ייצוא ל-PDF** - דוחות מעוצבים עם תמיכה בעברית
- **עיצוב אחיד** - כל הדוחות בעיצוב FIBI DSM

### 🔍 חיפוש מתקדם
- **חיפוש גלובלי** - חיפוש בכל העובדים, דרישות ומשימות
- **Autocomplete חכם** - הצעות בזמן אמת
- **פתיחה מהירה** - לחיצה על תוצאה פותחת את הפרטים

### 👥 ניהול צוותים
- **5 תצוגות**:
  - מנהל מדור (תצוגה מלאה)
  - צוות לוגאשי
  - צוות ספישווילי
  - צוות ריטל
  - צוות הויזמן

## 🚀 התקנה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- שרת HTTP פשוט (Python, Node.js, או כל שרת אחר)

### הרצה מקומית

1. **שכפול הפרויקט**
```bash
git clone https://github.com/YOUR_USERNAME/matafOS.git
cd matafOS
```

2. **הרצת שרת HTTP**

עם Python 3:
```bash
python3 -m http.server 8081
```

עם Node.js:
```bash
npx http-server -p 8081
```

3. **פתיחה בדפדפן**
```
http://localhost:8081
```

## 📖 שימוש

### 1. טעינת נתונים
1. לחץ על "דרישות" בכותרת העליונה
2. בחר קובץ Excel עם נתוני הדרישות
3. לחץ על "שעות" בכותרת העליונה
4. בחר קובץ Excel עם נתוני השעות

### 2. בחירת תצוגה
- בחר צוות מהרשימה הנפתחת בכותרת העליונה
- הדשבורד יתעדכן אוטומטית לפי הצוות הנבחר

### 3. חיפוש
- השתמש בשדה החיפוש הגדול מתחת לטאבים
- הקלד שם עובד, דרישה או משימה
- לחץ על תוצאה לפתיחת הפרטים

### 4. ייצוא נתונים
- לחץ על כפתור "אקסל" או "PDF" בכל טבלה
- הקובץ יורד אוטומטית למחשב

## 📁 מבנה הפרויקט

```
matafOS/
├── index.html              # עמוד ראשי
├── teams-structure.json    # הגדרת צוותים
├── css/
│   ├── styles.css         # עיצוב ראשי
│   └── fibi-dsm.css       # מערכת עיצוב FIBI
├── js/
│   ├── main.js            # נקודת כניסה ראשית
│   ├── config.js          # הגדרות מערכת
│   └── modules/
│       ├── data-processor.js    # עיבוד נתונים
│       ├── ui-renderer.js       # תצוגה
│       ├── file-handler.js      # טיפול בקבצים
│       ├── exporter.js          # ייצוא
│       ├── global-search.js     # חיפוש
│       ├── team-filter.js       # סינון צוותים
│       └── modal-manager.js     # ניהול חלונות
├── assets/                # פונטים
├── webfonts/             # אייקונים
└── README.md             # תיעוד זה
```

## 🛠️ טכנולוגיות

- **Vanilla JavaScript (ES6+)** - ללא תלות בספריות חיצוניות
- **Modular Architecture** - ארכיטקטורה מודולרית נקייה
- **SheetJS (xlsx)** - קריאת קבצי Excel
- **jsPDF + html2canvas** - יצירת PDF עם תמיכה בעברית
- **Chart.js** - גרפים וויזואליזציות
- **Font Awesome** - אייקונים

## ⚙️ תצורת צוותים

ערוך את הקובץ `teams-structure.json` להגדרת הצוותים:

```json
{
  "teams": [
    {
      "id": "team-id",
      "name": "שם הצוות",
      "manager": "שם המנהל",
      "description": "תיאור",
      "employees": ["מספר עובד 1", "מספר עובד 2"]
    }
  ]
}
```
